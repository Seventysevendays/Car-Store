## 二手车项目

------

前端：react、dva、antd组件
后端：nodejs
数据库：mongodb
打包工具：webpack

> * 完整的项目结构
> * 丰富的功能模块
> * antd高效开发
> * 前后端一气呵成

------

### 项目总结
> * 通过基础数据，使用mock.js完善模拟数据
> * 使用mongod，创建数据库，创建collection-car，并导入二手车的模拟数据
> * 使用node.js，mongoose链接数据库，创建后台接口
> * 安装react / dva / express / less相关依赖
> * webpack配置
> * 路由：dva/router
> * 包括：选车模块、用户模块、管理员模块、注册用户模块
